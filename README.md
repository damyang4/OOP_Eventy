# OOP_Eventy

## Github Link
[Eventy](https://github.com/damyang4/OOP_Eventy)

## How to start (Windows)
- Compile all source files dynamically into a single binary executable

>`g++ -std=c++17 src/main.cpp src/core/*.cpp src/commands/*.cpp src/services/*.cpp src/storage/*.cpp src/domain/*.cpp src/utils/*.cpp -o eventy`

- Run the compiled binary application

>`./eventy`
